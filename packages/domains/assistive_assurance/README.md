# Assistive Assurance

Phase 8 task 417 owns the backend evidence pipeline for material assistive change control.

This package turns regulatory and release-control evidence into deterministic records instead of loose document folders. It classifies assistive changes, routes IM1 RFC, SCAL, DTAC, DCB, DPIA, and medical-device reassessment triggers, pins assurance baselines, enforces independent approvals, proves rollback readiness, opens assurance freezes on drift, settles release actions against exact hashes, and exports governed evidence artifacts.
