Bounce-back and reopen recovery atlas

Same-shell recovery board for urgent return, routine reopen, and loop-risk escalation. Every variant stays anchored to the pharmacy workspace shell and the selected case frame.

PHC-2103

Urgent return

Direct professional route, monitored safety net, and non-calm patient copy.

  • Banner uses alert semantics.
  • Primary action returns to validation board.
  • Patient message preview is emitted and traceable.
PHC-2204

Routine reopen

Original request anchor stays visible and recovery continues in the same shell.

  • Open-original-request action is primary context support.
  • Return message preview is ready but not emitted.
  • Reopen diff shows calm-safe posture versus current recovery.
PHC-2215

Loop-risk escalation

Repeated returns force supervisor review and suppress patient messaging.

  • Escalation card stays harder to miss than routine chrome.
  • Auto-redispatch and auto-close remain blocked.
  • Contact-route repair stays explicit in the rail.
