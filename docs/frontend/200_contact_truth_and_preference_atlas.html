Contact truth source separation and repair visibility atlas

This atlas defines how authenticated account details separate NHS login claims, Vecells communication preferences, and optional external demographic rows without implying cross-source writes.

Source-of-truth comparison board

NHS login claims

View-only in Vecells and externally managed through NHS login.

  • Source: NHS login
  • Freshness: current session
  • Edit authority: NHS login only

Vecells preferences

Editable or reviewable communication behavior for Vecells only.

  • Source: Vecells preference
  • Freshness: patient reviewed
  • Edit authority: Vecells preference

PDS and GP demographics

External demographic rows when projected, never preference truth.

  • Source: external demographic
  • Freshness: projection-specific
  • Edit authority: external system
