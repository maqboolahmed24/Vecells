# 14 Visual Direction And Showcase Spec

        ## Visual Thesis

        Signal Atlas Live / Quiet Clarity: a restrained, evidence-first internal product language where one dominant operating moment stays visible, token law is obvious in the DOM, and trust cues outrank dashboard noise.

        ## Content Plan

        - hero: chosen stack, gateway split, design-contract bundle, Playwright posture
        - shell studio: one live shell preview with route morphing, selected-anchor preservation, and runtime markers
        - contract inspector: current gateway surface, query or mutation or live-channel contracts, cache posture, and bundle digest
        - closing proof: shell families, coverage counts, and publication law summary

        ## Interaction Thesis

        - one soft morph for route changes inside the same shell
        - one reduced-motion-safe emphasis change for anchor or anomaly selection
        - one subtle support-region promotion that never competes with the dominant action

        ## Shared System

        - max width `1440px`
        - `xs` and `sm` use 4 columns with 16px gutters
        - `md` uses 8 columns with 24px gutters
        - `lg` uses 12 columns with 24px gutters
        - `xl` uses 16 columns with 32px gutters
        - rail `72px` collapsed / `280px` expanded
        - support peek `320px`; support open `clamp(320px, 28vw, 400px)`
        - inline drawer `clamp(360px, 33vw, 480px)`
        - system sans only, no remote fonts
        - semantic color export fixed at the shared showcase palette from the prompt

        ## Showcase Shells

        - `Patient shell`: One calm mission frame for the current request
- `Workspace shell`: Queue and task canvas stay in one two-plane shell
- `Operations shell`: Runtime evidence map with one dominant anomaly and one dominant action region
- `Governance shell`: ScopeRibbon, ChangeEnvelope, ImpactPreview, ApprovalStepper, and ReleaseFreezeTupleCard share one continuity path

        ## Branding

        - One subtle Vecells monogram built from three connected nodes forming a restrained V.
        - Inline SVG only. No icon packs, remote assets, or clip-art medical motifs.
